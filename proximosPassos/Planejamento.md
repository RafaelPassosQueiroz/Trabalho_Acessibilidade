## Documento de Definições Iniciais (Planejamento do MVP)

Esta seção define o escopo inicial do projeto **Loja de Action Figures**, bem como a organização da equipe para o desenvolvimento do MVP. O objetivo é estruturar as primeiras entregas, garantindo clareza nas responsabilidades e alinhamento entre os integrantes.

### Escopo do MVP
O MVP terá como foco:
- Exibição de catálogo de action figures  
- Página de detalhes do produto  
- Carrinho de compras básico  
- Estrutura inicial de acessibilidade (A11y)  
- Integração inicial com backend e banco de dados
